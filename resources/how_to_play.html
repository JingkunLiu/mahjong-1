How to paly Mahjong

1. Introduction
Mahjong (NOT Mahjong Solitaire) is a game that originated in China. It is widely played in China and some Asian areas. Mahjong is a game of skill, strategy and calculation and involves a certain degree of chance.

There are many variants of Mahjong with different rules. To make it easy, this game use the most simple one. Any one who never touched Mahjong would learn to play it in one minutes. Besides, we redesigned some tiles to make it easy to recognize for people who don't know Chinese characters in Mahjong tiles.

3. How to win

Collect 4 sets of either "Pung", "Sheung" or "Kong" and must have a pair of same tiles known as "Jong" to win, The “Jong” must have point of 2 or 5 or 8.

  • Pung : Set of 3 identical tiles from any suits

  • Sheung: A run of 3 tiles from same suit (9, 1, 2 is not permissible)

  • Kong: 4 identical tiles from any suits

  • Jong: A pair of same tiles, must be of point 2, 5 or 8

4. Game rules

The game will start when all four players are ready.

  • Dealing: This is automatically done by game server. After dealing, the East ( first player who join the game ) has 14 tiles, the rest of the players have 13 tiles each. And it’s turn for east to discard.
  • Discard: The current player discard an unwanted tile face up into the middle, the tile may be useful for other players to make up a set. So other players may call this tile. If no one calls, the next player ( clockwise ) will pick up one tile from the Wall, and it’s turn for him/her to discard.
  • Call: If the unwanted tile is useful for one player to make up a set as in the following way:
    • Pung ( 3 identical tiles)

If player has 2 identical tiles in hand which match the unwanted tile, “Pung” can be called.

    • Kong ( 4 identical tiles)
  • If player has 3 identical tiles in hand which match the unwanted tile, “Kong” can be called. This is known as “Soft Kong”.
  • If player picks up an identical tile from the Wall that match the other 3 tiles in hand, “Kong” can be called. This is known as “Hard Kong”.

Note: when "Kong" is shouted, player should pick up another tile from the tail of Wall (This is automatically done by game server).

    • Hooo ( win the game)
  • If player needs the unwanted tile to win, “Hooo” can be called and we say the player who discard the unwanted tile made a "Fire".
  • If player picks up a tile and meet a win set, “Hooo” can be called. This is known as "Self Pick".

5. Scores

Scores are automatically calculated by server, the rules are as follow:

  1. If no one wins after tiles are used out, this round is draw, players’ score stay unchange.
  2. If a player wins not by "Self Pick", the player who made a "Fire" will give the winner 5 points. Note that winners may be more than one if serveral players need the same tile to win at the same time.
  3. If a player wins by "Self Pick", all three other players are losers and must give the him/her 5 points each.
  4. If a player made a "Soft Kong", the player who help him/her made this must give 5 points to him/her.
  5. If a player made a "Hard Kong", all three other players must give him/her 5 points each.
